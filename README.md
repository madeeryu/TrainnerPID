## Prerequisite
- Windows OS (10/newer).
- Anaconda for virtual environment.

## How to install
1. Install nmap from [official website](https://nmap.org/dist/nmap-7.95-setup.exe).
2. ```conda create -n <env_name> python=3.12 -y```
3. ```conda activate <env_name>```
4. ```pip install python-nmap```
